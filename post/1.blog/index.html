如何在Windows 10 使用Hugo搭建自己的博客

一、 安装Hugo

参考：https://bwaycer.github.io/hugo_tutorial.hugo/tutorials/installing-on-windows/

1.1 设置目录

需要一个位置来存储Hugo可执行文件，hugo程序包的内容bin（您构建的文件）和存放个人博客的内容Sites。

打开Windows资源管理器。 创建一个新文件夹：（C:\Hugo假设您希望Hugo在C驱动器上–它可以在任何地方使用。推荐还是存在放在D盘）

在Hugo文件夹中创建一个子文件夹：C:\hugo\bin。

在Hugo中创建另一个子文件夹：C:\hugo\sites。

1.2 下载地址

Hugo的GitHub地址：https://github.com/gohugoio/hugo/releases

找到目标文件：hugo_0.69.2_Windows-64bit.zip

下载得到一个zip的压缩文件，然后解压到C:\Hugo\bin中

1.3 设置环境变量

新建环境变量：

变量名 HUGO_HOME

变量值 C:\hugo

然在Path的环境变量中，新增 %HUGO_HOME%bin

1.4 验证是否安装成功

打开cmd，输入hugo version，显示以下内容，安装成功

1

Hugo Static Site Generator v0.69.2-EC9DCF30 windows/amd64 BuildDate: 2020-04-24T07:51:17Z

七、 如何更新博客

1

hugo new post/newblog.md

打开新建的newblog.md，编辑blog内容

或者，不需要新增，只对已存在的md文档进行修改

编辑后检测内容：cd 到blog根目录，然后输入以下命令：

1

hugo server -D

更新public静态页面，先cd到blog根目录，然后输入以下命令：

1

hugo --theme=even --baseUrl="https://daniel1n.github.io/" --buildDrafts

推送更新内容,cd到public目录下，然后输入以下命令：

1
2
3
4

git add .
git commit -m "Hugo Blog update"
git push -u origin master

总结

  1. 在window下安装，有些麻烦，下载hugo_0.69.2_Windows-64bit.zip，非常的慢
  2. 安装后，需要设置环境变量，才能在cmd中启动
  3. 设置主题很重要，默认是不安装主题的
  4. 在本地启动个人博客，检测博客是否正常在本地显示
  5. 远端的部署，需要先新增好repository
  6. 部署成功后，可以对博客进行新增和修改，实际只是在本地编辑好，更新public的内容，再次提交到远端即可
