什么是虚拟机？ 所谓的虚拟机，其本质其实也是一个程序。 但是这个程序，模仿了一台完整的主机，一旦进入了这个程序，就放佛进入了另一台独立的电脑似的。 这个“独立的电脑” 有其CPU,内存，硬盘，网络，ip地址等等 独立的资源。 这个所谓的 “独立的电脑” 就是虚拟机啦。

在这个虚拟机里安装一个Linux 操作系统，与我们花钱在 阿里云上买的 Linux 操作系统的使用，是一样一样的，所以通过在 虚拟机里的Linux ，我们就可以很好地锻炼自己的 Linux 操作技巧。

1 虚拟机软件

我们选择 VMware ，vmware 官网地址：https://www.vmware.com/

2 关闭Hiper-V

如果你是 win10, 那么自带了个虚拟机叫做 Hiper-V. 在安装 VMWare前要把他关闭了，否则会和VMWare有冲突。

键盘快捷键 win+i ，打开控制面板，点击程序 ，启动或关闭 Windows 功能 。

找到 Hyper-V, 如果其处于选中状态，把勾选去掉，然后点确认。

3.3 创建虚拟机

  1. 启动 VMWare， 点击创建新的虚拟机
  2. 选择典型，然后下一步
  3. 选： 稍后安装操作系统， 下一步
  4. 客户机操作系统选择: Linux(L)； 版本选择： CentOS 7 64 位； 下一步
  5. 虚拟机名称： CentOS 7 64 位 (2)，位置随机
  6. 使用默认的20G；. 默认的 将虚拟磁盘拆分成多个文件； 下一步
  7. 点击自定义硬件
    1. 内存：2G
    2. 使用默认的1个处理器，1核就行了。 不过： 虚拟化 Intel VT-x/EPT 或者 AMD-V/RVI(V) 必须勾上
    3. 新 CD/DVD （IDE ) ：选择IOS镜像文件
    4. 网络适配器 ：桥接模式
    5. 完成
  8. 点击 开启此虚拟机
  9. 鼠标点击进去，然后用键盘选中 Install CentOS 7后回车
  10. 等了好一会儿后，出现安装界面：选择中文和简体中文
  11. 安装信息摘要 ：
    1. 安装位置：什么都别动。。。 点击完成
    2. 网络和主机名：右上角，点成打开，然后点击完成
    3. 点击 开始安装
  12. 配置：设置root的密码（个人习惯）
  13. 重启 ，等待
  14. 重启后，就出现重启界面：
    1. 登录账号为：root
    2. 密码为个人设置的密码
  15. 查看ip地址，输入 ifconfig

安装完成后的设置

1 SHH客户端的软件

终端链接有Xshell和SecureCRT

远程文件传输WinSCP

2 端口开放设置

可以设置常用的：3360到8888的端口，其他主机就可以访问了

1
2
3
4

[root@Centos ~]# iptables -I INPUT -p tcp --dport 3360:8888 -j ACCEPT
[root@Centos ~]# service iptables save
[root@Centos ~]# service iptables restart
[root@Centos ~]# iptables -nL #查看规则表，看下添加成功否

3 MYSQL的安装

可以使用dockers安装mysql 5.7

也可以直接用yum wget 直接安装

  1. 安装wget: yum -y install wget

  2. 安装vim：yum -y install vim

  3. 获取mysql社区版

    1
2
3

    cd /tmp
wget http://repo.mysql.com/mysql-community-release-el7-5.noarch.rpm  
rpm -ivh mysql-community-release-el7-5.noarch.rpm

  4. 通过yum进行安装

    1

    yum install mysql mysql-server mysql-devel -y

  5. 启动 :

    1

    systemctl start mysql.service

  6. 验证 :

1

netstat -anp|grep 3306

  1. 设置mysql密码

    通过Yum安装的mysql的管理员账户是没有密码的，这里通过命令设置其密码为 admin

    1

    mysqladmin -u root password admin

  2. 登陆验证 : mysql -u root -p admin

  3. 通过本地mysql可视化软件，访问远程mysql

    1. 第一步，开放端口，添加3360端口iptables访问规则：

      1
2
3
4

      [root@Centos ~]# iptables -I INPUT -p tcp --dport 3360 -j ACCEPT
[root@Centos ~]# service iptables save
[root@Centos ~]# service iptables restart
[root@Centos ~]# iptables -nL #查看规则表，看下添加成功否

    2. 第二步，允许mysql被其他主机访问：

      1
2
3

      mysql -u root -p admin
mysql> GRANT ALL PRIVILEGES ON *.* TO 'root'@'%' IDENTIFIED BY 'youpassword' WITH GRANT OPTION;
mysql> FLUSH PRIVILEGES;

    3. 最后，就可以通过 ifconfig，查看ip地址，输入ip、端口号（3360）、用户名、密码，连接mysql
